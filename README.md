# ClientRestAPI
A reference example for Spring mvc Rest API

# Prerequisites:
Java 8
Maven
Apache Tomcat

# How to deploy
Run maven command mvn tomcat:deploy or mvn tomcat:redeploy

# Application URL
http://localhost:8080/ClientRestAPI/client
